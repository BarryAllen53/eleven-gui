# Eleven GUI 1.0.1

## Highlights

- Added an Inno Setup installer pipeline for Windows
- Added optional code-signing support for release artifacts
- Separated portable packaged mode from installed packaged mode through a `.portable` marker
- Moved installed-build configuration and generated files to per-user Windows locations

- `ElevenGUI-1.0.1-win64.zip`

## Notes

- Portable builds keep configuration, cache, and outputs next to the executable through the `.portable` marker.
- Installed builds use `%APPDATA%\ElevenGUI` for the saved API key and `%LOCALAPPDATA%\ElevenGUI` for cache and generated output.
- Local signing can be done with a self-signed development certificate for internal testing.
- Public trusted distribution still requires a real code-signing certificate.

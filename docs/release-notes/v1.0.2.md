# Eleven GUI 1.0.2

## Highlights

- Fixed long-text generation playback so merged audio includes the full content
- Added model-aware language selection with safer fallback behavior for unsupported language combinations
- Added progress updates every 20 seconds with estimated completed and remaining characters/words
- Expanded TTS output format options to include modern WAV, OPUS, PCM, and telephony profiles

- Long text synthesis uses chunked generation and reliable merge behavior for multi-part output.

# Eleven GUI 1.0.0

## Highlights

- Accessible Windows desktop client for ElevenLabs built with Python 3.11 and PySide6
- Workspace overview for subscription, credits, cloning access, and model readiness
- Voice Hub with My Voices, Cloned Voices, and Shared Library flows
- Studio workflows for text-to-speech and speech-to-speech with auto-play, replay, regenerate, and download
- Clone Lab support for Instant Voice Clone and Professional Voice Clone flows
- Keyboard shortcut coverage, list-first navigation, and screen-reader-oriented UI behavior across the app
